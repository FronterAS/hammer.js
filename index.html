Getting started

Hammer is easy to use. Just create an instance and bind the events. By default it supports all the standard gestures you would expect.

var mc = Hammer(myElement);
mc.on("swipeleft swiperight", mySwipeHandler);

By default it supports the tap, doubletap, pan, swipe, hold, pinch and rotate gestures. The default touch-action property is set to pan-y, so horizontal scrolling is being prevented, but vertical scrolling is still possible.
